# Security Policy

## Supported Versions

Currently, only the latest release of `teaql-forge-rs` is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within `teaql-forge-rs`, please do NOT open a public issue. 
Instead, please send an e-mail to our security team via [security@teaql.dev](mailto:security@teaql.dev). All security vulnerabilities will be promptly addressed.
